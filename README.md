# Fast Food Spinner 🍔

A fun React app that helps you decide where to eat when you're indecisive. Spin the wheel and let fate choose your next fast food destination!

## Features

- **Spin the Wheel** – Colorful wheel with your favorite fast food restaurants and a satisfying spin animation
- **Customize Restaurants** – Add or remove restaurants from the wheel
- **Spin History** – See a log of past results with timestamps

## Quick Start

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).
